01 ‚Äî Introduction

Who, what and how?

Namche is a personal AI project built around the idea of mindful, grounded technology. Named after the Sherpa gateway town of Namche Bazaar, it represents a base camp for exploring what AI assistants can be.

The project's AI bot, Tashi, embodies a philosophy rooted in Buddhist principles without spirituality: clarity, presence, compassion, and pragmatic action.

The brand architecture follows the Mycelia framework, creating a flexible system where Namche acts as the parent brand and Tashi as its primary product entity.

02 ‚Äî Brand concept

Organic meets Technical

Following the Mycelia brand system, the core design principle merges organic and technical forms. Round, fluid shapes intertwine with angular, robust shapes to create a hybrid visual language.

For Namche, this translates to the calm clarity of mountain landscapes meeting the precision of technology. Stone meets sky. Ancient paths meet modern tools.

03 ‚Äî Logo

Logo Symbol

The logo symbol depicts the letter "N" constructed within the 3x3 block grid system. Each block is separated by one eighth of its own square dimension.

The "N" is formed by a left vertical bar (three squares), a diagonal connection using quarter-circle transitions, and a right vertical bar. This creates the characteristic organic-technical hybrid form.

Blocks can be filled with one of three basic shapes: a full square, a quarter circle, or a half circle. Shapes can rotate and expand to neighbouring blocks.

Basic shapes

Square / Quarter circle / Half shape
04 ‚Äî Logo with type

Logo symbol with type

The logo type is set in Inter Bold with custom letter-spacing applied. The positioning of the two elements is based on the symbol's dimension.

The logo is always positioned in corners of the canvas. Logo symbol and type can be positioned separately in different corners.

It is possible to use the logo symbol without the logo type but not the other way around.

05 ‚Äî Colors

Color System

Colors play an important role across the brand system. Each entity gets its own color scheme.

Primary and secondary colors define the color scheme. Create a brand's color scheme as follows:

1. Choose a primary color
2. Define a dark and light shade
3. Choose a secondary color
4. Define a dark and light shade

For the "full" hue, choose a color with brightness value between roughly 50 and 80 in the HSB color wheel, so that there is enough difference for the dark and light shades.

Width of medium / 28 * 26

Elements can be multiplied to create interesting patterns. Vertically the elements can be positioned freely. Horizontally they are positioned along the columns of the logo element.

08 ‚Äî Composition with type

Applying Type

Type is always set in black or white depending on the background. Always aim for high contrast.

The positioning of elements is handled quite freely. Let the background pattern guide you in order to create interesting compositions.

09 ‚Äî Color variations

Color Variation

The system allows for plenty of variation when applying color. Add color by:

1. Choose a background color (e.g. gold/full)
2. Choose a color for the pattern from the opposite color (e.g. blue/full)
3. Apply a linear gradient to the pattern (e.g. 0% to 100% transparency)

As guidance, use the same shade type for both colors: e.g. blue/light and gold/light.
